WHAT IS THIS?
-------------
A powerful multi-threaded folder deletion tool with a modern dark UI.
Delete stubborn folders that Windows won't let you remove.


HOW TO USE:
-----------
1. Run FolderForceDeleter-Portable.exe
2. Click BROWSE to select a folder
3. Click FORCE DELETE
4. Confirm the deletion
5. Watch the progress bar

That's it!


ADMIN MODE (RECOMMENDED):
-------------------------
For best results, run as administrator:

  Right-click FolderForceDeleter-Portable.exe
  → Select "Run as administrator"

This enables:
  ✓ Deletion of system folders
  ✓ Taking ownership of locked files
  ✓ Removing protected files
  ✓ Full force-delete capabilities


FEATURES:
---------
• Multi-threaded deletion (uses all CPU cores)
• Real-time progress tracking with percentage
• Status bar showing operation status
• Modern dark theme UI
• Handles locked, readonly, and system files
• No installation required - runs from anywhere


PERFORMANCE:
------------
• 1,000 files:    1-2 seconds
• 10,000 files:   5-10 seconds  
• 100,000 files:  30-60 seconds

Speed increases with more CPU cores!


SYSTEM REQUIREMENTS:
--------------------
• Windows 10 or 11 (64-bit)
• ~200 MB free RAM
• Administrator rights recommended


WARNING:
--------
⚠️  This tool PERMANENTLY deletes folders and all their contents.
⚠️  There is NO UNDO or recycle bin.
⚠️  Always double-check before clicking FORCE DELETE.

Use with caution!


PORTABLE APP:
-------------
• No installation needed

STATUS BAR:
-----------
Bottom of the window shows:
  • Current operation status
  • Admin privileges (YES/NO)
  • Real-time progress during deletion


TROUBLESHOOTING:
----------------
Windows blocks the file?
  → Right-click → Properties → Unblock → Apply

Some files won't delete?
  → Run as administrator
  → Windows protects some system files

App won't start?
  → Make sure Windows 10/11 64-bit
  → Check antivirus isn't blocking it


WHAT HAPPENS WHEN DELETING:
----------------------------
1. SCANNING:  Counts all files in folder
2. DELETING:  Multi-threaded parallel deletion
3. CLEANUP:   Removes empty directories
4. COMPLETE:  Shows results

Progress shows:
  • Percentage complete
  • Items deleted count
  • Current operation
